# � Floating Music Player (FMP)

**Floating Music Player** is a minimalist, background-aware media controller that lives in your system tray and floats on your screen.

---

## ✨ Features

- **Ultra-Minimal UI**: A compact 120x48 bar that expands to a 120x96 dual-bar controller.
- **Background Recognition**: Automatically detects when system audio is playing and updates the tray status.
- **Smart Controls**: Functional Play/Pause and Volume Up/Down buttons that control system-wide media.
- **Always-on-Top**: Stays visible while you work, but small enough to never get in the way.
- **Tray Integration**: Hide to tray on close, with dynamic tooltips showing current playback status.
- **Frameless & Draggable**: No bulky title bars; click and drag anywhere on the bar to move it.

## 🚀 Tech Stack

- **Runtime**: [Electron 30](https://www.electronjs.org/)
- **Frontend**: [React 18](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Integration**: Native PowerShell for Windows media & volume control.

## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/itshakemd/floating-music-player.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build the application**:
   ```bash
   npm run build
   ```
